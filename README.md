  <h1>Conversor de Monedas</h1>
  <p>
    Este programa es un conversor de monedas desarrollado en <strong>Java</strong>. Permite convertir entre diferentes monedas (Dólar estadounidense, Peso argentino, Real brasileño y Peso colombiano) usando tasas de cambio obtenidas en tiempo real desde una API externa.
  </p>

  <h2>Características principales</h2>
  <ul>
    <li>Conversión entre <code>USD</code>, <code>ARS</code>, <code>BRL</code> y <code>COP</code>.</li>
    <li>Interfaz por consola.</li>
    <li>Obtención automática de tasas de cambio actualizadas.</li>
    <li>Manejo de errores en la consulta a la API.</li>
  </ul>

  <h2>Uso</h2>
  <div class="pasos">
    <ol>
      <li>Ejecuta el programa.</li>
      <li>Selecciona la opción de conversión.</li>
      <li>Ingresa la cantidad a convertir.</li>
      <li>Visualiza el resultado en pantalla.</li>
    </ol>
  </div>

  <h2>Requisitos</h2>
  <div class="requisitos">
    <ul>
      <li>Java 11 o superior.</li>
      <li>Conexión a internet para consultar la API de tasas de cambio.</li>
    </ul>
  </div>
